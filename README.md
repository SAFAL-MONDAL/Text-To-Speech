
#  Text-to-Speech Converter  
*A web application that converts text into natural-sounding speech using modern browser APIs.*  

![Demo Screenshot](https://safalportfolio.vercel.app/) <!-- Replace with actual project screenshot -->

## ✨ Features  
- **Real-time speech synthesis** from text input  
- Multiple voice options (gender, language, pitch control)  
- Download audio as MP3/WAV files  
- Responsive design (works on mobile & desktop)  
- Dark/Light mode toggle  

## 🚀 Live Demo  
Experience it live: [https://text-to-speech-app.vercel.app/](https://text-to-speech-app.vercel.app/)
